# Superpowers Workflow Artifacts

This directory contains [Superpowers](https://github.com/obra/superpowers/) generated planning artifacts for Verbatim.

These files are useful for understanding accepted goals, reviewed designs, and future implementation plans. They should not replace the durable source-of-truth architecture docs under `docs/architecture`, `docs/phases`, `docs/tooling`, `docs/observability`, or `docs/parity`.

## Layout

| Path     | Purpose                                                                               |
| -------- | ------------------------------------------------------------------------------------- |
| `specs/` | Accepted design specs and project charters produced through Superpowers brainstorming |
| `plans/` | Future implementation plans produced from accepted specs                              |

## Current Entry Point

Start with:

- `specs/2026-06-27-verbatim-rewrite-design.md`

That spec records the project goals and non-goals, then links to the durable architecture, phase, tooling, observability, and parity documentation.
